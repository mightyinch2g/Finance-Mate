# Finance Mate

Eigenständiges Python-Desktopprogramm als Finance-/ERP-Light-System für KMU.

## Aktueller Stand
- v0.1 Startgerüst
- Desktop-App mit tkinter
- SQLite als Startdatenbank
- Layout angelehnt an FiBu Mate
- kleinere Kopfzeile
- kleinere Fußzeile
- ohne Logos

## Start
```bash
python FinanceMate.py
```
